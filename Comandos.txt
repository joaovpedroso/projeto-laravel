------ ****** INSTALAÇÃO E CONFIGURAÇÃO LARAVEL ****** ------

-INSTALAÇÃO DO LARAVEL VERSAO 5.5
	*composer create-project --prefer-dist laravel/laravel NOME_DO_PROJETO "5.5.*"

- CRIAÇÃO DE AUTENTICAÇÃO PARA ACESSO
	*PHP ARTISAN MAKE:AUTH

- ATUALIZAÇÃO DA TABELAS DO LARAVEL NO BD
	*PHP ARTISAN MIGRATE 

- REMOVER ERRO DAS MIGRATES
	*DIRETORIO APP -> PROVIDERS -> APP SERVICE PROVIDER.PHP
	ADICIONAR AS LINHAS:

		NAMESPACES
			use Illuminate\support\Facades\Schema;
		
		E DENTRO DA FUNÇÃO BOOT
			 Schema::defaultStringLength(191);

- ATIVAÇÃO DO NODE JS
	*NPM INSTALL




------ ****** COMANDOS ****** ------

- CRIAÇÃO DE MODEL
	*PHP ARTISAN MAKE:MODEL NOME_DA_MODEL

- CRIAÇÃO DE MIGRATION
	*PHP ARTISAN MAKE:MIGRATION NOME_DA_MIGRATION

- CRIAÇÃO DE CONTROLLER COM OS METODOS PADRÕES
	*php artisan make:controller NOME_DO_CONTROLLER --resource

- RODAR O PROJETO
	*PHP ARTISAN SERVE

- ATUALIZAÇÕES EM 2º PLANO
	*NPM WATCH
